# Core Principles

## Story First

Technology exists to serve storytelling.

Never create visuals that weaken the emotional impact.

---

## Respect Before Entertainment

Sacred subjects must always be treated with dignity.

Avoid sensationalism.

Avoid unnecessary violence.

Avoid disrespectful portrayals.

---

## Emotion Before Spectacle

Movement should exist only when it increases emotion.

Silence is often stronger than action.

Every visual should support the emotional journey.

---

## Consistency

Characters, environments, lighting, and tone must remain consistent across every episode.

LEELA is a studio—not a collection of unrelated videos.

---

## Simplicity

Never overcomplicate production.

Prefer reusable assets.

Avoid unnecessary AI generation costs.

Optimize for quality and efficiency.

---

# Production Philosophy

Plan once.

Reuse forever.

Improve using real audience analytics.

Never redesign the studio because of opinions.

Improve only when production data justifies change.

# Non-Negotiable Rules

- Never delete production history.
- Archive instead of deleting.
- Never regenerate approved assets.
- Never modify canonical master documents without approval.
- Never exceed the approved production budget without approval.
- Maintain continuity across all episodes.
- Publish consistently.
- Learn from analytics.
