# SECTION 5 — THE EDITING FORMULA

---

## Making Still Images Cinematic (Zero AI Cost):

### TECHNIQUE 1: Slow Zoom
```
Start: 100% scale
End: 105-108% scale
Duration: 4-6 seconds
Direction: Center-weighted (zoom toward subject's eyes)
Effect: Creates forward movement, intimacy, approaching
Use for: Portraits, emotional moments, revelations
```

### TECHNIQUE 2: Ken Burns Pan
```
Start: Frame left
End: Frame right (or vice versa)
Movement: 3-5% horizontal shift
Duration: 5-8 seconds
Effect: Scanning a scene, journey, exploration
Use for: Landscapes, wide shots, environments, groups
```

### TECHNIQUE 3: Parallax (2-Layer)
```
Method: Separate subject from background in CapCut (mask tool)
Foreground: Moves slightly MORE than background
Background: Moves slightly LESS
Effect: Simulates depth, 3D feel from a 2D image
Use for: Character in environment (forest, palace, river bank)
```

### TECHNIQUE 4: Fog/Dust Overlay
```
Asset: Fog_Ground_Level.mp4 or Dust_Floating.mp4
Blend mode: Screen or Add
Opacity: 10-25%
Effect: Adds LIFE. The image breathes. Something moves.
Use for: Every indoor scene, every forest scene, every morning scene
```

### TECHNIQUE 5: Light Rays
```
Asset: Light_Rays_Golden.mp4
Blend mode: Screen
Opacity: 15-30%
Position: Upper corner (motivated by window/sun/divine source)
Effect: Volumetric depth, atmosphere, sacred feeling
Use for: Temple, prison (through crack), forest (through canopy), divine moments
```

### TECHNIQUE 6: Rain Overlay
```
Asset: Rain_Overlay_Heavy.mp4
Blend mode: Screen
Opacity: 30-50%
Effect: Instant monsoon. Image becomes a rainy scene.
Use for: Journey scenes (Vasudeva walking — combine rain overlay + image of empty street)
```

### TECHNIQUE 7: Depth Blur (Post)
```
Method: Duplicate image layer. Blur bottom layer. Mask sharp area on top.
Effect: Simulated shallow depth of field on any image
Use for: Making AI images look more cinematic (foreground sharp, background soft)
```

### TECHNIQUE 8: Color Grade Shift
```
Method: Keyframe color temperature over duration of the clip
Start: Warm (golden)
End: Cool (blue) — or vice versa
Effect: Emotional shift WITHIN a single held image
Use for: Scene transitions, mood changes, day-to-night implied shifts
```

### TECHNIQUE 9: Sound Design Sync
```
Method: Sync ambient sound hits with visual elements
Example: Bell toll synced with cut between images
Example: Thunder crack synced with lightning overlay flash
Example: Footstep sounds over a still image of a path = "walking"
Effect: Sound makes the viewer BELIEVE there's motion even when there isn't
```

### THE RULE:
> Any image + slow zoom + fog overlay + ambient sound + narrator voice
> = a cinematic scene indistinguishable from video at YouTube compression.

# SECTION 7 — PRODUCTION DECISION CHECKLIST

---

## Before Generating ANY Asset, Ask:

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  "Can this scene work as a STILL IMAGE          │
│   with editing motion + sound design?"          │
│                                                 │
│        YES → Generate IMAGE (Category B)        │
│         │                                       │
│        NO → "Does it absolutely REQUIRE         │
│              visible motion to tell              │
│              the story?"                        │
│                                                 │
│              YES → Generate VIDEO (Category A)  │
│              NO  → Use image + overlay (C)      │
│                                                 │
└─────────────────────────────────────────────────┘
```

### Specific Decision Guide:

| Scene Type | Image or Video? | Reasoning |
|-----------|----------------|-----------|
| Character talking | IMAGE | Voice carries it. Face held = powerful. |
| Character walking | IMAGE | Slow pan + footstep sounds = implied walking |
| River flowing | IMAGE + water ripple overlay | Overlay gives life |
| River PARTING (miracle) | VIDEO | Water defying gravity MUST be motion |
| Rain falling | IMAGE + rain overlay | Rain overlay = instant monsoon |
| Fire/torch | IMAGE + fire overlay | Flicker overlay = living flame |
| Divine appearance | VIDEO | The moment God appears must MOVE |
| Transformation | VIDEO | Shape-shifting requires motion |
| Sleeping characters | IMAGE | Static = perfect for sleep |
| Battle/action | IMAGE sequence (fast cuts) | Quick cuts between images = energy |
| Landscape | IMAGE + Ken Burns | Pan creates the cinematic feel |
| Close-up emotion | IMAGE + slow zoom | Zoom into eyes = maximum intimacy |
| Crowd scene | IMAGE | One well-composed image = enough |
| Animals | IMAGE | Peacock/cow still = fine. Only exception: if peacock DANCES |
| Flying/floating | VIDEO | Defying gravity requires motion |
