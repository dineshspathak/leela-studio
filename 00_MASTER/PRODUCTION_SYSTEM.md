## 1. PRODUCTION SOP (Standard Operating Procedure)

### The 90-Minute Video Machine — 7 Phases

```
┌─────────────────────────────────────────────────────────────┐
│  PHASE 1: RESEARCH & SCRIPT          (25 min)               │
│  PHASE 2: VOICEOVER GENERATION       (10 min)               │
│  PHASE 3: VISUAL GENERATION (Veo 3)  (20 min)               │
│  PHASE 4: EDITING & ASSEMBLY         (20 min)               │
│  PHASE 5: THUMBNAIL CREATION         (5 min)                │
│  PHASE 6: SEO & METADATA             (5 min)                │
│  PHASE 7: QUALITY CHECK & UPLOAD     (5 min)                │
│                                                              │
│  TOTAL: 90 MINUTES → 1 PREMIUM VIDEO                        │
└─────────────────────────────────────────────────────────────┘
```

### Phase 1: Research & Script (25 min)
```
Step 1.1 — Pick topic from Content Calendar          (1 min)
Step 1.2 — Research: Scriptural source + facts       (10 min)
Step 1.3 — Write script using Script Template        (12 min)
Step 1.4 — Review: Hook strong? Lesson clear?        (2 min)
```

### Phase 2: Voiceover (10 min)
```
Step 2.1 — Clean script (remove directions)          (2 min)
Step 2.2 — Run through Google AI Studio (Gemini TTS)     (5 min)
Step 2.3 — Listen & verify pronunciation             (3 min)
```

### Phase 3: Visual Generation (20 min)
```
Step 3.1 — Break script into scenes (storyboard)     (3 min)
Step 3.2 — Generate visuals using Veo 3 prompts      (12 min)
Step 3.3 — Download & organize clips                 (5 min)
```

### Phase 4: Editing (20 min)
```
Step 4.1 — Import audio + visuals into CapCut        (2 min)
Step 4.2 — Sync visuals to narration                 (10 min)
Step 4.3 — Add transitions, music, effects           (5 min)
Step 4.4 — Add intro + outro                         (2 min)
Step 4.5 — Export (1080p, 30fps)                     (1 min)
```

### Phase 5: Thumbnail (5 min)
```
Step 5.1 — Select best frame or generate image       (2 min)
Step 5.2 — Apply thumbnail template in Canva         (2 min)
Step 5.3 — Export 1280×720 PNG                       (1 min)
```

### Phase 6: SEO (5 min)
```
Step 6.1 — Write title (using SEO template)          (1 min)
Step 6.2 — Write description (using template)        (2 min)
Step 6.3 — Add tags + hashtags                       (1 min)
Step 6.4 — Set timestamps                            (1 min)
```

### Phase 7: QC & Upload (5 min)
```
Step 7.1 — Run Brand Checklist (15 checkboxes)       (2 min)
Step 7.2 — Upload + set thumbnail + schedule         (3 min)
```

## 8. VOICEOVER TEMPLATE

### Voice Configuration (LOCKED):
```
Platform:       Google AI Studio (Gemini TTS)
Voice:          Gemini native audio (Male, warm storytelling, calm spiritual tone)
Language:       hi-IN (Natural Indian accent, age 35-50)
Pace:           Slow cinematic pacing
Output:         WAV or MP3
```

### Script Formatting Rules for TTS:

```
DO:
✅ Use full stops (।) for natural pauses
✅ Use commas for breath pauses
✅ Write numbers with commas: "10,000" not "10000"
✅ Use "..." for dramatic pauses (TTS adds slight pause)
✅ Keep sentences under 30 words (natural breathing)
✅ Write Hindi in Devanagari script for best pronunciation

DON'T:
❌ Don't use emojis in TTS script
❌ Don't use markdown formatting (**bold**, etc.)
❌ Don't use English words unless necessary (Hinglish sounds off)
❌ Don't write [directions] in the TTS text
❌ Don't make paragraphs too long (split at natural breaks)
```

### Voiceover Script Format:
```
[INTRO - Narrator]
[5 sec silence for intro animation]

[HOOK]
उस रात, जब पूरी पृथ्वी अंधकार में डूबी थी... एक जेल की कोठरी में, एक दिव्य 
प्रकाश प्रकट हुआ।

[PAUSE - 1 sec]

[ACT 1]
ये कथा है भगवान श्री कृष्ण के जन्म की। वो रात... जो सृष्टि के इतिहास में 
सबसे अद्भुत रात मानी जाती है।

...

[CLOSING]
और यही सीख हमें मिलती है इस कथा से... कि अंधकार कितना भी गहरा हो, प्रकाश 
अवश्य आता है।

[PAUSE - 1 sec]

अगली कहानी में, कृष्ण ने गोकुल में ऐसा कुछ किया जो सुनकर आप चौंक जाएंगे... 
जय श्री कृष्ण।

[12 sec silence for outro]
```

### Audio Post-Processing:
1. Normalize volume to -14 LUFS (YouTube standard)
2. Remove any clicks/artifacts
3. Add 0.5s silence at start and end
4. No compression or effects on voice (keep natural)

## 9. EDITING TEMPLATE (CapCut)

### Project Settings:
```
Resolution:     1920 × 1080 (16:9)
Frame Rate:     30 fps
Color Space:    Rec. 709
Export:         H.264, High Quality, 30 Mbps
```

### Timeline Structure:
```
TRACK 1 (top):     Text overlays / Lower thirds
TRACK 2:           Visual effects / Particles / Transitions
TRACK 3:           Main visuals (AI clips + stock footage)
TRACK 4:           Background color/gradient (safety)
─────────────────────────────────────────────────────
AUDIO 1:           Narrator voiceover (PRIMARY)
AUDIO 2:           Background music (bansuri/ambient)
AUDIO 3:           Sound effects (bell, conch, nature)
```

### Standard Sequence:
```
[0:00 - 0:05]    INTRO (pre-made asset, drop in)
[0:05 - 0:15]    HOOK scene (most dramatic visual)
[0:15 - 8:30]    MAIN STORY (synced visual + audio)
[8:30 - 9:30]    LESSON (calming visuals, wisdom tone)
[9:30 - 9:42]    OUTRO (pre-made asset, drop in)
```

### Transition Library (use ONLY these):
| Transition | When to Use | Duration |
|-----------|-------------|----------|
| Cross dissolve | Default between scenes | 1.5s |
| Light flash (white) | Divine moments, miracles | 0.5s |
| Fade to black | Scene change, time skip | 2.0s |
| Zoom blur | Dramatic reveal | 0.8s |
| Particle wipe (gold) | Entering sacred space | 1.5s |

**NEVER use:** Spin, glitch, page turn, bounce, or any "fun" transitions.

### Color Grading Preset:
```
Shadows:        Push toward deep blue (#1e3a8a)
Midtones:       Neutral with slight warmth
Highlights:     Golden warmth (+10 amber)
Saturation:     +5% (slightly more vivid than reality)
Contrast:       +10% (cinematic depth)
Vignette:       Subtle dark corners (15%)
```

### Text/Lower-Third Style:
```
Font:           Noto Sans Devanagari Medium
Size:           Medium (readable on mobile)
Color:          White (#f8fafc) with 50% opacity black shadow
Position:       Bottom center, safe margin from edges
Animation:      Fade in 0.5s → hold → Fade out 0.5s
```

### Audio Mixing Levels:
```
Narrator Voice:     0 dB (reference level)
Background Music:   -18 dB to -22 dB (barely audible, atmospheric)
Sound Effects:      -8 dB to -12 dB (noticeable but not jarring)
Silence gaps:       Keep 0.5-1s between major sections
```

## 16. THE LEELA CINEMATIC RULE (LOCKED)

Before designing any storyboard, writing any visual prompt, or choosing a production method for any scene, this decision framework must be applied automatically.

### STEP 1: Determine Emotional Intensity

For every scene in the script, evaluate its narrative purpose and emotional weight to assign a level:

*   **Level 1 — Information Only**: Scene establishing facts, context, or landscape transitions.
*   **Level 2 — Conversation**: Characters communicating, narrating backstory, or reciting scriptures.
*   **Level 3 — Emotion**: Quiet character moments, reaction shots, sadness, yearning, or devotional prayer.
*   **Level 4 — Major Emotional Moment**: Climax points, high-tension decisions, or sudden revelations.
